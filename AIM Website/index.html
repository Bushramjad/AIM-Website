URDUISTICS

Our Urdu NLP Library

We are excited to announce the launch of our Urdu NLP library that we call Urduistics. Urduistics contains a wide variety of NLP implementations in Python for processing and analyzing Urdu text. Using Urduistics, developers can build analytic solutions for data in Urdu, with production quality code within minutes.
